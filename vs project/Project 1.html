Html Tag Report

Introduction

HTML (HyperText Markup Language) serves as the foundation of web development, allowing developers to structure and present content on the web efficiently. This report provides an in-depth exploration of over 60 HTML tags, categorized into sections for document structure, text formatting, lists, forms, multimedia, and more. Through this report, the aim is to develop a better understanding of the key components and their applications in creating dynamic and well-structured web pages. Each section is supplemented with examples to illustrate practical implementation and expected output, ensuring a comprehensive learning experience.

HTML Tags

Document Structure


        <html>: The root element of the HTML document.
        <head>: Contains metadata like title, styles, or scripts.
        <title>: Sets the title of the web page (shown on the browser tab).
        <body>: Contains the content of the web page.

Text Formatting


        <h1>: to <h6>: Headings (h1 is the largest, h6 the smallest).
        <p>:Paragraph.
        <b> or <strong>: Bold text.
        <i> or <em>: Italicized text.
        <u>: Underlined text.
        <span>: Inline container for styling.
        <blockquote>: Block-level quotation.
        <q>: Inline quotation.
        <em>: Emphasis.
        <mark>: Highlighted text.
        <abbr>: Abbreviation or acronym.
        <address>: Contact information.
        <time>: Represents time.
        <sub>: Subscript text.
        <sup>: Superscript text.
        <small>: Smaller text.

Forms and Input

            <form>: Form container.
            <input>: Input field.
            <label>: Describes an <input> element.
            <textarea>: Multiline text input.
            <button>: Clickable button.
            <select>: Dropdown list.
            <option>: Option in a <select> list.
            <optgroup>: Groups <option> elements inside a <select>.
            <fieldset>: Groups form controls and labels.
            <legend>: Describes the content of a <fieldset>.
            <datalist>: Contains a set of <option> elements for use with <input list>.
            <output>: Represents the result of a calculation or user action.
